# Mue API
Source code for the Mue API.

## Installation
### Requirements
* [Git](https://git-scm.com)
* [Node.js](https://nodejs.org)

### Setup
1. Clone the repository with ``git clone https://github.com/mue/api``
2. Enter the directory and install NPM dependencies with ``npm i`` (or ``yarn`` if you prefer)
3. Run ``node src/setupdatabase.js`` in order to setup the database (SKIP IF USING EXISTING DATABASE FILE in ``src``)
4. Fill out ``config.json`` with your server information
5. Finally, run the server in src with ``node index.js``
